# GifCapture
GifCapture is a free animated [.GIF](https://en.wikipedia.org/wiki/GIF) recorder and converter for Mac, written in [Swift](https://developer.apple.com/swift/).

## Features
- Recording iOS Simualtor
- Recording Screen
- Converting MOV to GIF
